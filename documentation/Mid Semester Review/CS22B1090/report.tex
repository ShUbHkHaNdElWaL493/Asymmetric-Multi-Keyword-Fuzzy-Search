% Abstract
\abstract

Enabling keyword search directly over encrypted data is a promising paradigm for the efficient use of outsourced sensitive information in the cloud environment. Although various techniques have achieved significant advancements in multi-keyword fuzzy search through the use of locality-sensitive hashing (LSH) and Bloom filtering techniques to prevent the expansion of the index file, various critical challenges are associated with the existing techniques for their scalable implementation. First of all, the basic techniques are based on symmetric-key cryptography, which is not suitable for key management and distribution in a multi-user environment. Second, a strong trade-off is associated with the basic LSH technique between false positive rates and false negative rates, resulting in a reduced recall value of the search results if the fuzzy keywords are included as a part of the search query.

In order to overcome these challenges, this report suggests a sophisticated privacy-preserving multi-keyword fuzzy search system that employs the combination of asymmetric cryptography and Multi-Probe LSH. By transforming the underlying matching algorithm from a matrix-based operation to an Inner Product Predicate Encryption (IPE) paradigm, this proposed system enables the owners to safely outsource the data using a public-key-based mechanism, allowing the exact search capabilities to be delegated to the users without the leakage of the master key. Additionally, the replacement of the traditional LSH technique with Multi-Probe LSH enables the intelligent extension of the search space to the adjacent buckets that contain the highest probability of containing the nearest neighbors. Such a replacement of the underlying algorithm eliminates the false negative rate of the traditional LSH-based search systems without the expansion of the underlying hash table.
